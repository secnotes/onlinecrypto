# 在线加解密工具

基于 Crypto.js 的纯前端加密解密工具。所有操作均在浏览器本地完成，数据不会上传至任何服务器。

## 功能特性

### 对称加密
- **AES** - 高级加密标准，目前最流行的对称加密算法，安全可靠
- **DES** - 数据加密标准（已过时，建议使用 AES 替代）
- **TripleDES** - 三重 DES，比 DES 更安全但速度较慢
- **Blowfish** - 快速安全的对称加密算法，密钥长度可变
- **Rabbit** - 高性能流密码算法，适合实时加密
- **RC4** - 流密码算法，简单快速
- **RC4Drop** - RC4 改进版本，安全性更高

### 哈希算法
- **MD5** - 128 位哈希（不推荐用于安全场景）
- **SHA-1** - 160 位哈希（逐渐被淘汰）
- **SHA-224** - SHA-2 系列，224 位哈希
- **SHA-256** - SHA-2 系列，256 位哈希，广泛使用
- **SHA-384** - SHA-2 系列，384 位哈希
- **SHA-512** - SHA-2 系列，512 位哈希，安全性高
- **SHA-3** - 新一代哈希算法，基于 Keccak 结构
- **RIPEMD-160** - 160 位哈希

### HMAC
- 基于哈希的消息认证码，用于验证数据完整性和真实性

### 编码工具
- **Base64** - Base64 编码/解码
- **Hex** - 十六进制编码/解码
- **Latin1** - ISO-8859-1 编码/解码
- **UTF-8** - UTF-8 字节编码查看
- **综合转换** - 自动识别输入格式并转换为所有其他格式

综合转换支持的输入格式：
- UTF-8 字符串（普通文本）
- Hex 十六进制（如 `48656c6c6f`）
- Base64
- Inline 格式（如 `\x48\x65\x6c\x6c\x6f`）
- C 数组格式（如 `unsigned char data[] = { 0x48, 0x65, 0x6C };`）
- Python 格式（如 `data = [ 0x48, 0x65, 0x6C ]`）

### 密钥派生
- **PBKDF2** - 基于密码的密钥派生函数
- **EvpKDF** - OpenSSL EVP 密钥派生函数

## 高级选项

### 加密模式
- CBC（密码分组链接）- 推荐
- ECB（电子密码本）- 不推荐，不安全
- CFB（密码反馈）
- OFB（输出反馈）
- CTR（计数器模式）

### 填充方式
- PKCS#7（最常用）
- 零填充
- 无填充
- ISO/IEC 9797-1
- ANSI X.923

### 密钥格式
- 字符串（UTF-8）
- Hex 十六进制
- Base64
- Latin1

## 安全与隐私

- 所有加密解密操作均在浏览器本地完成
- 数据不会上传至任何服务器
- 支持自动生成随机密钥
- 支持暗色/亮色主题切换
- 支持中英文双语（根据浏览器设置自动检测）

## 使用方法

直接在浏览器中打开 `index.html` 即可使用，无需安装或部署服务器。

## 技术栈

- 纯 HTML/CSS/JavaScript
- [Crypto.js](https://github.com/brix/crypto-js) 用于加密操作
- 响应式设计，支持桌面和移动设备
